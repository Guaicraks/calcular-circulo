Este proyecto es una aplicación en C++ que permite calcular diferentes propiedades de un círculo a partir del radio ingresado por el usuario.
El programa utiliza un menú interactivo implementado con do...while y switch, permitiendo realizar múltiples cálculos sin reiniciar la ejecución.

Funcionalidades:

- Calcular el área del círculo
- Calcular el perímetro (circunferencia)
- Modificar el valor del radio
- Salir del programa

Fórmulas utilizadas

Área:
A = π × r²

Perímetro:
P = 2 × π × r

Se utilizó una constante:
const double PI = 3.1416;
Para garantizar mayor precisión en los cálculos.

Tecnologías utilizadas
- Lenguaje: C++
- Compilador: Dev C++ (compatible con cualquier compilador estándar de C++)
- Paradigma: Programación estructurada

Conceptos aplicados:
- Variables int y double
- Uso de const
- Estructura repetitiva do...while
- Estructura condicional switch
- Operaciones matemáticas
- Entrada y salida con cin y cout
